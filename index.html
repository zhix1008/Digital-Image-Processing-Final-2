CIFAR-10 模型展示平台

這是一個可直接部署到 GitHub Pages 的前端模型測試網站。使用者可以切換不同模型、 上傳圖片或點選範例圖片，讓瀏覽器直接載入 ONNX 模型並執行推論。

模型切換與測試控制
GoogleNet CIFAR-10 基準模型
檔案：googlenet_cifar10.onnx
特性：準確率相對較低，適合與較佳模型做比較。
ResNet18 CIFAR-10 較佳模型
檔案：resnet18_cifar10.onnx
特性：準確率較好，預期辨識表現更穩定。

請上傳一張圖片進行測試

支援一般圖片格式（JPG / PNG / WEBP）。系統會自動縮放為 32×32 後送入模型。
