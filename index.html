CIFAR-10 ONNX 模型展示平台

本網站可直接部署於 GitHub Pages，使用者可切換 GoogleNet 與 ResNet18 模型， 上傳任意尺寸圖片或點選範例圖片進行推論。系統會依模型輸入設定自動縮放圖片， 並於瀏覽器中使用 ONNX Runtime Web 執行推論。
